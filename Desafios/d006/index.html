Tags em HTML

Em HTML5 podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso(tooltips simples).

  • Parágrafos
  • Títulos
  • Textos com realce forte(negrito)
  • Textos com ênfase(itálico)
  • Citações
    • Citações curtas
    • Citações longas(em bloco)
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas
    • Listas numeradas
    • Listas com demarcadores
  • Hiperlinks
